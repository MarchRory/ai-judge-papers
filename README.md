## 前端技术栈

| 分类             | 选型                               |
| ---------------- | ---------------------------------- |
| 框架             | Vue3 全家桶                        |
| 状态管理         | Pinia                              |
| 语言             | TypeScript                         |
| 后台模板         | Acro-Design-pro-vue                |
| 组件库           | Arco-Design-vue                    |
| CSS 框架         | UnoCSS, 这种规模的项目用原子化方便 |
| 图标集           | iconify 里挑一个, 感觉 tabler 还行 |
| 第三方 hooks 库  | VueUse                             |
| 网络请求库(待选) | axios                              |
